
# Project Title

Shopping list - react app

## Run Locally

Clone the project

```bash
  git clone https://github.com/ZygaDev/shoppinglist
```

Go to the project directory

```bash
  cd shoppinglist
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```


## Installation

Install project with npm to used on server

```bash
npm install
npm run prod 
```
